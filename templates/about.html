About Atom Manga Reader

Atom Manga Reader (AMR) is an application designed to read manga on Kobo eReaders.

This application was developed for the Kobo ereaders, which do not have an inbuilt manga reader for reading manga from online sources. It was designed to be easy to install and use.

The app was built entirely with python for the back end. I relied on html for the front end, to provide buttons and images for navigating and viewing.

You can find more details about the app on mobileread and GitHub.
